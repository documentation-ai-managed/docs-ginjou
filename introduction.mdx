## What is Ginjou?

Ginjou provides a centralized hub for your project documentation. You create, organize, and collaborate on docs using an intuitive interface powered by MDX. Manage everything from API references to user guides without leaving your workspace.

<Callout kind="info">
Ginjou supports real-time editing, version history, and seamless integration with your Git workflow.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://ginjou.com` with your email or GitHub.
  </Step>
  <Step title="New Workspace" icon="plus">
    Click "New Workspace" and name it for your project, like `my-app-docs`.
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    
````mdx
<CodeGroup tabs="MDX,Markdown">
```mdx
---
title: My First Doc
description: Hello world.
---
## Welcome
Start writing here.
```
```markdown
# My First Doc

Hello world.
```
</CodeGroup>
````

    Paste into the editor and publish.
  </Step>
  <Step title="Publish & Share" icon="globe">
    Hit publish to generate a shareable URL like `https://docs.ginjou.com/my-app-docs`.
  </Step>
</Steps>
